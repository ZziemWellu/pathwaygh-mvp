# PathwayGH - AI-Powered Career Guidance for Ghanaian Students

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.115+-green.svg)](https://fastapi.tiangolo.com)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://reactjs.org)

**PathwayGH** helps Ghanaian students discover the right career paths based on their interests, subjects, and WASSCE grades.

## Quick Start

```bash
# Clone and setup
git clone https://github.com/ZziemWellu/pathwaygh-mvp.git
cd pathwaygh-mvp

# Backend (Python)
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000

# Frontend (React - new terminal)
cd frontend
npm install
npm run dev
Visit: http://localhost:5173

Features
🔍 Career Search

🎓 University Programme Finder

📊 WASSCE Eligibility Checker

🤖 AI Career Quiz
